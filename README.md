# KakaoTalk Chat Viewer 💬

[![Live Demo](https://img.shields.io/badge/demo-online-brightgreen.svg)](https://panewcolour.github.io/kakaotalk-chat-viewer/)

A lightweight, browser-based local viewer for exported KakaoTalk chat logs. 
This tool allows you to read your exported `.txt` chat files in a familiar, interactive interface without uploading any data to a server.

## 🚀 [Visit Live Site](https://panewcolour.github.io/kakaotalk-chat-viewer/)

## ✨ Key Features

* **Privacy First:** All parsing and rendering happen locally in your browser. No data is sent to any server.
* **Smart Parsing:** Supports both Korean and English KakaoTalk export formats (PC and Mobile).
* **Interactive Calendar:** Navigate through years and months to find specific dates with active chat logs.
* **Powerful Search:** Full-text search with keyword highlighting, match counters, and smooth scrolling.
* **Personalization:** "Set as Me" feature to align your own messages to the right, just like the actual app.
* **Responsive Design:** Fully optimized for both desktop and mobile viewing.

## 🛠 Tech Stack

* **HTML5 / CSS3**
* **JavaScript (Vanilla JS)**
* **Tailwind CSS** (via CDN)
* **Lucide-style SVG Icons**

## 📖 How to Use

1.  **Export Chat:** In KakaoTalk, go to a chat room -> Settings -> Export Messages (Save as Text Only).
2.  **Open Viewer:** Click the [Live Demo](https://panewcolour.github.io/kakaotalk-chat-viewer/) link.
3.  **Upload:** Drag and drop your `.txt` file onto the upload area or click to select the file.
4.  **Explore:** Use the sidebar to search for keywords or navigate using the calendar.

## 🔒 Privacy & Security

User privacy is the top priority. This application is a **static web page**. 
* It **does not** have a backend server.
* It **does not** collect, store, or transmit your chat logs.
* Once the page is loaded, it can even function offline.
